# C++ Object-Oriented Programming Projects

This repository contains a collection of C++ projects developed as part of an Object-Oriented Programming module. The projects demonstrate core C++ principles, including object-oriented design, algorithm development, simulation systems, and application development using external frameworks.

---

## 1. Cryptocurrency Trading Simulator & Algorithmic Trading Bot

### Overview
Developed a cryptocurrency trading simulator in C++ featuring a modular object-oriented architecture. The system was extended with an algorithmic trading bot that autonomously executes trading decisions based on a custom prediction function.

The bot evaluates simulated market conditions and generates buy/sell/hold decisions, which are executed within the trading environment.

### Key Features
- Built a simulated cryptocurrency market environment using C++ and OOP principles
- Designed and implemented trading system components (assets, market logic, and trade execution)
- Developed a custom `predict()` function to analyse market conditions
- Tested trading strategies under dynamic simulated conditions

### Skills Demonstrated
- C++ (OOP, class design, modular programming)
- Algorithm design and rule-based decision systems
- Simulation development
- Data-driven logic and state management
- Problem-solving and independent implementation of trading strategies

## 2. DJ Mixing Application (JUCE Framework)

### Overview
Developed a desktop DJ mixing application using the JUCE C++ framework. The application provides audio playback, mixing functionality, and a graphical user interface.

The project was later extended with additional features and improvements, demonstrating the ability to enhance and maintain a growing codebase.

### Key Features
- Built an interactive audio player using the JUCE framework
- Implemented audio file loading and playback controls
- Developed a graphical user interface for user interaction
- Added mixing functionality for audio tracks
- Extended base functionality with additional features and UI improvements

### Skills Demonstrated
- C++ application development
- JUCE framework (GUI and audio systems)
- Event-driven programming
- Audio processing fundamentals
- Software extension and refactoring
